create

Runnable create(int uniqueNumber)
Creates a Runnable that should be run iterationCount() number of times. This method is called ConcurrencyTester.NR_OF_CONCURRENT_RUNNERS times so the total number of executions will be iteratationCount * NR_OF_CONCURRENT_RUNNERS. Any RuntimeException or Error thrown from the created runnable will be caught and propagated through ConcurrencyTester.verify(RunnableFactory, long, TimeUnit). For performance and test-harness reasons the variables created based on uniqueNumber should be saved and reused by the repeated runs.

Parameters:
uniqueNumber - should be used to differentiate results made from different threads to increase the odds of detecting thread-safety issues.

iterationCount

int iterationCount()
Number of times to run each of the created Runnables.
