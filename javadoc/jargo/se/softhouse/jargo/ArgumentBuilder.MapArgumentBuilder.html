public static final class ArgumentBuilder.MapArgumentBuilder<K,V>
extends ArgumentBuilder<BUILDER,T>
    An intermediate builder used by ArgumentBuilder.asPropertyMap(). It's mainly used to switch the T argument of the previous builder to Map<K, T> and to indicate invalid call orders.

 String[] arguments = {"--number", "1", "--number", "2"};
 List<Integer> numbers = integerArgument("--number").repeated().parse(arguments);

 If you want to combine ArgumentBuilder.repeated() with a specific ArgumentBuilder.arity(int) then call
 ArgumentBuilder.arity(int) before calling ArgumentBuilder.repeated():
 
 
         
 String[] arguments = {"--numbers", "1", "2", "--numbers", "3", "4"};
 List<List<Integer>> numberLists = integerArgument("--numbers").arity(2).repeated().parse(arguments);
 assertThat(numberLists).isEqualTo(asList(asList(1, 2), asList(3, 4)));
