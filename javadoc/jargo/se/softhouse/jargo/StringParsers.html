Class StringParsers

  • java.lang.Object
    • se.softhouse.jargo.StringParsers

  • @Immutable
public final class StringParsers
extends Object
     Gives you static access to implementations of the StringParser interface.
 All methods return Immutable parsers.
 Most methods here even return the same instance for every call.
 If you want to customize one of these parsers you can use a ForwardingStringParser.
 
 By default most parsers return zero or otherwise
 sane defaults. The only exception being enumParser(Class) which returns null instead
 of the first enum constant available.

      • enumParser

        @CheckReturnValue
@Nonnull
public static <T extends Enum<T>> StringParser<T> enumParser(Class<T> enumToHandle)
         A parser that uses Enum.valueOf(Class, String) to parse input strings.
 
 Case sensitivity note: First a direct match with the input value in upper case is
 made, if that fails a direct match without converting the case is made,
 if that also fails an ArgumentException is thrown. This order of execution is
 based on the fact that users typically don't upper case their input while
 java naming conventions
 recommends upper case for enum constants.
        Default value: null is used as StringParser.defaultValue() and not any of the enum values in enumToHandle
        Parameters:
        enumToHandle - the Class literal for the Enum to parse strings into
        Returns:
        a StringParser that parses strings into enum values of the type T
