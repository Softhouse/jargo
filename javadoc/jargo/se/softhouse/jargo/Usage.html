public final class Usage
extends Object
implements Serializable

Responsible for formatting usage texts for CommandLineParser.usage() and Arguments.helpArgument(String, String...). Using it is often as simple as: 

 
 System.out.print(CommandLineParser.withArguments(someArguments).usage());
 
 
 Sorts Arguments in the following order:
 
    
   
  1. indexed arguments without a variable arity
    2. 
         
  2. By their first name  in an alphabetical order
    3. 
         
  3. The remaining indexed arguments that are of variable arity
    4.

Appends usage to target, just like printOn(Appendable) but for a PrintStream instead. The reason for this overload is that Appendable declares IOException to be thrown while PrintStream handles it internally through the use of the PrintStream.checkError() method.
