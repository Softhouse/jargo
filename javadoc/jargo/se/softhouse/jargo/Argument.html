@Immutable
public final class Argument<T>
extends Object

 Represents a supported Argument for a command line invocation.
 
 Arguments are created with the static methods in Arguments or with a custom ArgumentBuilder and then
 used by the CommandLineParser to parse strings (typically from the command line).

parse


@Nullable
public T parse(String... actualArguments)
        throws ArgumentException




Parses command line arguments and returns the value of this argument.

 This is a shorthand method that should be used if only one Argument is expected.
 If several arguments are expected use CommandLineParser.withArguments(Argument...)
 instead. 

 Locale: Locale.US is used to parse strings / print usage. To use another
 Locale you'll need to use CommandLineParser.withArguments(Argument...)
 instead.





Parameters:
actualArguments - the arguments from the command line

Returns:
the parsed value from the actualArguments

Throws:

ArgumentException - if actualArguments isn't what this
             argument expects
