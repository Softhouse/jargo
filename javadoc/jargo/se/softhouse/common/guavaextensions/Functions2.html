      • compound

        @Nonnull
public static <T> Function<T,T> compound(Function<T,T> first,
                                 Function<T,T> second)
        Runs several Functions in the same order as they are given as arguments here.
        If one of the Functions makes T Immutable, make sure to pass it in last as it's hard to modify an Immutable value. This works exactly like Functions.compose(Function, Function) except that if first doesn't do anything, second is returned directly instead.
        Parameters:
        first - a Function
        second - another Function
        Returns:
        a merged Function
