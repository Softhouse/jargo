se.softhouse.common.numbers

Class NumberType<T extends Number>

  • java.lang.Object
    • se.softhouse.common.numbers.NumberType<T>
  • Type Parameters:
    T - the subclass of Number
    Direct Known Subclasses:
    NumberType.UnlimitedNumberType

    public abstract class NumberType<T extends Number>
extends Object
    A class that exposes static fields (and functions), such as Integer.MAX_VALUE and Integer.MIN_VALUE, for subclasses of Number in an object oriented way.
    Note: As Double and Float are very hard to use right, their counterparts are not available here.
